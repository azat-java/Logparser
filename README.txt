LogParser

Программа позволяет спарсить логи из файла csv и сохранить их в БД.

Запуск приложения:

1) Распаковка архива;
2) Создание в локальной БД новой БД 'logparser';
3) В файле Logparser\src\main\resources\hibernate.cfg.xml:
	3.1) Ввести логин и пароль от БД в строках 10-11;
	3.2) Поменять в строке 21 'update' на 'create'. После первого запуска программы, убедившись в создании столбцов таблицы, поменять значение вновь на 'update'.
4) Запустить приложение из терминала, в качестве аргумента указав название файла с логами. Название тестового файла: data/logs_example.csv
